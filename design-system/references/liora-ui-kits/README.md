# Liora UI Kits — Visual References Only

These files are visual references.

They are not the production design system.

The production source of truth is:

`design-system/foundations/`

Use these HTML files to study:
- layout
- density
- hierarchy
- product-specific UI patterns
- operator workflows
- messaging patterns
- guest guide composition

Do not use:
- `colors_and_type.css` as production tokens
- Inter as the final font decision
- legacy variables such as `--bg-1`, `--fg-1`, `--accent-700` in production code

Production code must use Product Design Foundations tokens.
