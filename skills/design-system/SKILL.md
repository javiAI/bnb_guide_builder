# SKILL · Design system

## Objetivo

Implementar la UI según la dirección reconciliada de v3:

- premium hospitality
- editorial SaaS
- utilitario silencioso

## Reglas

- labels visibles en español
- fondo claro y superficies limpias
- un CTA primario por bloque
- nada de dashboards densos ni ornamento gratuito
- consistencia visual entre módulos
